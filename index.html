What Is Bootstrap?

Bootstrap is one of the most popular libraries that web developers use, while new web development methods don't need to reuse every code on each component, and orgainising the website has became easer with the help of inernal libraries such as Bootstrap.

Bootstrap is one of the most powerfull that assenble HTML, CSS, asd JavaScript to build websites, or web apps. Basicly, it helps to build responsive websites on different screens, including mobile phones, and Ipads. this library is Free, and opensource on GitHub.

Why Bootstrap?

Ready Classes

reusable classes, ready to be used, you can find them on the original website.

Ready Compoment

precoded component ready to be used on your webstie, such as cards, navbar........

Framework

it unites the codein method between programmers, because it uses the same class names, and function.

Fast

it makes the whole development process easier.

Responsive

it has its own layout method that divides any screen into 12 columns, so you can place your component fast, and as wanted, istead of spending much time making your own one, and it helps with responsive so you can specify how many columns are being reserved to each component at several breakpoints.
